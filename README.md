Overview

An AI-powered system that predicts stock price movements by combining historical stock data with news sentiment analysis.

This project enhances traditional stock prediction models by incorporating market sentiment, providing more informed and realistic predictions.

Features
 Multi-index stock data processing
 Technical indicators (RSI, MACD)
 News sentiment analysis using NLP
 Machine learning-based prediction model
 End-to-end data pipeline
 Feature engineering for financial data

Tech Stack
 Language: Python
 Libraries: pandas, numpy, scikit-learn, tensorflow/pytorch
 NLP: nltk, textblob, transformers
 Visualization: matplotlib, seaborn

Workflow
 1) Collect stock and news data
 2) Preprocess and clean the dataset
 3) Perform feature engineering
 4) Apply sentiment analysis on news
 5) Train machine learning model
 6) Predict stock trends

Features Used
 Open, High, Low, Close (OHLC)
 Volume
 Daily Returns
 RSI (Relative Strength Index)
 MACD (Moving Average Convergence Divergence)
 Sentiment Score
